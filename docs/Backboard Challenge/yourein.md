Every submission must include a YouTube demo link. You can set the video to public or unlisted. Either is fine.

Video Name Format:

Backboard.io Spring Challenge 2026 - [Project name]

Length:

Aim for 4–5 minutes
Please don’t go longer than 8 minutes
What to show:

The actual demo of your project (most important)
Briefly: what it does and who it’s for
Briefly: how you’re using Backboard under the hood
On camera or not?

Being on camera is optional and more than welcome.

3. The submission form (mandatory)
To be judged, you must fill out the submission form properly and include your YouTube demo link. We’ll ask for:

Your project name
Your YouTube demo link
Repo link (GitHub/GitLab/etc.) if you have one
Short description of what it does
How you’re using Backboard (a few sentences is enough)
Which category/ies you’re aiming for (checkboxes)
Comments/Feedback
If any of the required fields are missing or obviously incomplete, we can’t judge the project.
